# experiment-doc Skill Evaluation

## Test Case 1: Build Experiment from Idea
**Input:** "I want to test if changing our homepage headline from features to outcomes increases trial signups"
**Expected Output:**
- Hypothesis statement
- Success metrics and targets
- Experiment design (control/variant)
- Sample size calculation
- Analysis plan
- Rigor score 70+

## Test Case 2: Audit Existing Experiment
**Input:** "Review this experiment doc (paste doc)"
**Expected Output:**
- Rigor score breakdown
- Specific failures flagged
- Recommendations for improvement
- Rewritten hypothesis if needed
- Go/no-go decision

## Test Case 3: Scorecard Mode
**Input:** "Score these 5 experiment ideas and tell me which to run first"
**Expected Output:**
- Each idea scored on rigor framework
- Ranked by score
- Effort vs impact assessment
- Top recommendation with rationale
