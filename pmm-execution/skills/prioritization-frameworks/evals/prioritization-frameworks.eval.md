# prioritization-frameworks Skill Evaluation

## Test Case 1: RICE Scoring
**Input:** "Score these 3 feature launches using RICE: (1) Enterprise dashboard, (2) Mobile app, (3) API v2"
**Expected Output:**
- RICE scores for each (Reach, Impact, Confidence, Effort)
- Ranked order with justification
- Recommendations on which to prioritize
- Sensitivity analysis on scoring assumptions

## Test Case 2: ICE vs RICE Comparison
**Input:** "Which framework should I use to prioritize our GTM backlog?"
**Expected Output:**
- Framework comparison
- Recommendation based on team size, complexity, data availability
- Example scoring using both methods
- When to use each

## Test Case 3: Launch Tier Assignment
**Input:** "Help me decide if this feature deserves a Tier 1 launch or Tier 3"
**Expected Output:**
- Tier definitions (T1-T4)
- Scoring against tier criteria
- Clear tier assignment with rationale
- GTM investment guidance per tier
