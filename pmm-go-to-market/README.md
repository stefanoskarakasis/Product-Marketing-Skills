# PMM Go-to-Market

Strategic planning and execution for product launches, GTM programs, and market expansion.

## Skills (2)

### go-to-market-strategy
Assigns launch tier (T1-T4) and generates strategic briefs with messaging and channels. Self-learns from past launches stored in brain Section 7.

**Use when:**
- Planning a product/feature launch
- Pricing change or expansion
- Need to scope GTM initiative

**Example prompts:**
- "We're launching SSO integration — what tier?"
- "Run full GTM strategy for analytics dashboard"
- "Help me scope this feature launch"

### workflow-orchestrator
Chains 15+ PMM skills for complete GTM workflows. Runs positioning → competitive → strategy → execution in one prompt.

**Use when:**
- Running full product launch (6-12 weeks)
- Quarterly PMM cycle refresh
- Competitive program
- Market entry strategy
- Positioning refresh

**Example prompts:**
- "Run Q3 PMM quarterly cycle"
- "Launch workflow for new market vertical"
- "Competitive program for top 3 competitors"

## Quick Start

New to this collection?
1. Set up your PMM brain: `/product-marketing-context`
2. Run a launch: `go-to-market-strategy` + workflow to scope GTM
3. Build positioning and competitive intel to feed GTM strategy
