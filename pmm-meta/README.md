# pmm-meta

Meta skills that operate on the skill system itself. Install alongside any PMM plugin
to improve skill quality and compound learnings across sessions.

## Skills (3)

- **meta-learn** — Captures post-session patterns, routes them to the correct knowledge files, and compounds intelligence across all skills over time.
- **meta-review** — Audits any SKILL.md against SKILL-SPEC v2.0.0 — 19-point checklist with prioritised fixes.
- **meta-verify** — Second-pass quality check on T1 skill output before delivery. *(Coming soon)*

## Commands (5)

- `/pmm-meta:learn` — Run post-session learning extraction for any skill.
- `/pmm-meta:learn-history` — Show what's been captured in the knowledge base.
- `/pmm-meta:learn-review` — Audit knowledge base for stale hypotheses and rules.
- `/pmm-meta:meta-review` — Run a 19-point spec review on a named skill.
- `/pmm-meta:fix` — Generate copy-paste content to fix a specific failing spec check.
